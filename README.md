
# Quit Smoking PWA

Quit Smoking PWA is a **Progressive Web Application (PWA)** designed to help individuals quit smoking by tracking progress, providing motivational resources, and offering an interactive and user-friendly interface.

---

## Features

- **Personalized Progress Tracker**  
  Track your journey with custom metrics like smoke-free days, money saved, and health improvements.


- **Offline Support**  
  As a PWA, the app works offline, ensuring accessibility anytime and anywhere.

- **Lightweight Design**  
  Built with a focus on performance and simplicity using **TypeScript**, **CSS**, and **JavaScript**.

---

## Tech Stack

The app is built with the following technologies:

- **TypeScript** (96.8%): Provides type safety and better tooling support.
- **CSS** (2.1%): Used for styling the application with custom and responsive designs.
- **JavaScript** (1.1%): Handles dynamic behavior and interactivity.

---

## Installation

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/harshmehta9/quit-smoking-pwa.git
   ```
2. Navigate to the project directory:
   ```bash
   cd quit-smoking-pwa
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```

---

## How to Use

1. Open the app in your browser or install it on your device as a PWA.
2. Set your quit-smoking goals and track your progress daily.
4. Access the app offline for uninterrupted support.
